# Gestor de Tareas 

Aplicación web para organizar tareas académicas, desarrollada con Flask y desplegada en la nube.

## Características
- 📝 Crear/eliminar tareas
- 🚨 Prioridades (Urgente, Normal, Baja)
- 📊 Gráfico de progreso

## Tecnologías
- Python 3.11
- Flask
- SQLite/PostgreSQL
- Bootstrap 5
